Keysight E5270B - Precision IV Analyzer (SMU)

Install

Install the instrument using the Manage Instruments button of NOMAD-CAMELS.
The PyPi package can be found here and can be installed via

pip install --no-cache-dir --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple <driver_name>
