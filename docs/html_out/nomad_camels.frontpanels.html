build_tree(search_text='', search_tag='')

Builds the tree of devices. First it clears the tree and then iterates through all available devices in self.device_dict. If search_text is given, only devices whose name includes the string in search_text are added to the tree. If search_tag is given, only devices that have the exact tag given by search_tag are added to the tree.

Parameters:

  • search_text – (Default value = ‘’)

  • search_tag – (Default value = ‘’)
